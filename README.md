# 🍷 Wine Classification Dashboard using PCA & Random Forest

## 📌 Project Overview

This project is a Machine Learning web application developed using **Python** and **Streamlit**. It classifies different types of wine using the Wine dataset from Scikit-learn. The project uses **Principal Component Analysis (PCA)** for dimensionality reduction and **Random Forest Classifier** for prediction.

---

## 🚀 Features

* Interactive Streamlit Dashboard
* Wine Type Prediction
* Principal Component Analysis (PCA)
* PCA Scatter Plot
* Scree Plot
* Correlation Heatmap
* Dataset Overview
* Easy-to-use Interface

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Joblib

---

## 🤖 Machine Learning Workflow

1. Load Wine Dataset
2. Data Preprocessing
3. Feature Scaling (StandardScaler)
4. PCA (Principal Component Analysis)
5. Train Random Forest Classifier
6. Save Model using Joblib
7. Deploy using Streamlit
